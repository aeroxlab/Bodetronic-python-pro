# Bodetronic Python Pro - Fotos y Panel Cliente

Incluye:
- Productos con 1 o más fotos de referencia.
- La primera foto queda como principal.
- Casillero/lista debajo del selector para confirmar fotos cargadas.
- Botón "Subir otra foto".
- Comparación visual por cámara usando todas las fotos registradas.
- Diseño de panel cliente con logo, fondo, colores, textos, botones, sponsor, información visible y vista previa.
- Publicidad/Sponsors con imágenes, videos, links, tamaño, ubicación y estilo tipo banner/franja/ticker.

## Render

Build Command:

```bash
pip install -r requirements.txt
```

Start Command:

```bash
uvicorn main:app --host 0.0.0.0 --port $PORT
```
